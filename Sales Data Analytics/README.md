# Sales Data Analytics Dashboard (Interactive Dashboard Using Tableau)

## Project Overview

This project presents an interactive Sales Dashboard built using Tableau to analyze sales performance, revenue trends, and customer behavior. The dashboard enables stakeholders to quickly understand business performance and make data-driven decisions.

## Dataset Used
- <a href="https://github.com/Sujataj266/Tableau/blob/main/Sales%20Data%20Analytics/Sales%20Data.xls">Sales-Dataset</a>
- <a href="https://github.com/Sujataj266/Tableau/blob/main/Sales%20Data%20Analytics/hexmap.xlsx">Hexmap-Dataset</a>

## Dashboard Features

1. KPI Cards
- Total Sales
- Total Profit
- Profit Margin
2. Visual Analysis
- Sales Trend Over Time
- Sales by Region
- Top Products by Sales
- Category-wise Performance
3. Interactive Elements
- Filters by Region, Category, and Date
- Drill-down for detailed insights
